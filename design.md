# Cyber Falcon - Design System

## Brand Identity
- **Name**: Cyber Falcon
- **Tagline**: "Precision. Protection. Performance."
- **Vibe**: Dark, cyber-futuristic, tech-forward, premium

## Color Palette
- **Background**: Deep black (#050510) with subtle blue undertone
- **Surface**: Dark navy (#0a0a1a), Card bg (#0d1117)
- **Primary Neon**: Cyan (#00f0ff) — main accent
- **Secondary Neon**: Electric purple (#a855f7)
- **Tertiary**: Hot pink/magenta (#ff0080) — for alerts/CTAs
- **Text Primary**: White (#f0f0f0)
- **Text Secondary**: Muted gray (#8892a4)
- **Border**: White at 8% opacity
- **Gradient**: Cyan → Purple for key highlights

## Typography
- **Display/Headlines**: "Orbitron" (Google Fonts) — geometric, techy
- **Body**: "Rajdhani" (Google Fonts) — clean, futuristic readability
- **Code/Mono**: "JetBrains Mono" — for technical flair

## 3D & Animation Strategy
- Three.js powered 3D hero with floating geometric shapes
- Parallax scroll effects on sections
- GSAP-powered scroll animations (staggered reveals)
- Hover 3D card tilt effects (vanilla CSS transforms)
- Particle background effects
- Glowing neon borders and pulsing elements

## Layout
- Full-width sections with max-w-7xl content containers
- Generous vertical spacing (py-24 to py-32)
- Asymmetric grids for visual interest
- Overlapping elements with z-index layering

## Components Style
- Cards: Glass-morphism with backdrop blur, neon borders on hover
- Buttons: Gradient fills with glow effects
- Icons: Lucide icons with neon glow
- Inputs: Dark with subtle borders, glow on focus

## Sections
1. **Hero**: 3D Three.js scene + animated headline + CTA
2. **Services**: 3D rotating cards for each service
3. **About**: Parallax sections with company story
4. **Blog**: Grid of blog preview cards
5. **Contact**: Form with backend submission
6. **Admin**: Dashboard for managing inquiries and blog
